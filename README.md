# Financial Analysis Using Excel

## Project Overview
This project demonstrates the use of Microsoft Excel financial functions for analyzing loans, investments, and financial decision-making scenarios.

## Objectives
- Understand key financial concepts
- Learn Excel financial functions
- Compare financial alternatives
- Improve decision-making through financial analysis

## Features
- Present Value (PV) Analysis
- Net Present Value (NPV) Analysis
- Internal Rate of Return (IRR)
- EMI Calculation
- Loan Duration (NPER)
- Investment Comparison

## Tools Used
- Microsoft Excel
- Financial Functions
- Charts and Graphs

## Key Insights
- Excel simplifies financial analysis.
- PV and NPV help compare financial options.
- EMI analysis explains loan repayment structures.
- NPV is often more reliable than IRR for investment evaluation.

## Applications
- Loan Planning
- Investment Analysis
- Home Loan Decisions
- Business Project Evaluation
